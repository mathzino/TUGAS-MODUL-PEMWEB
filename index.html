Teknik Publish Website

Publish web adalah cara untuk mempubikasikan web statis ataupun dinamis yang sudah kita buat agar dapat dilihat oleh user melalui browser. Agar dapat diakses oleh user untuk sebuah webste, kita memerlukan Domain dan Hosting.

Bagi anda yang belum mengetahui apa itu Domain. Singkat certa Domain adalah sebuah nama alamat yang nantinya akan digunakan user untuk mengakses, melihat atau menampilkan halaman web kita. Berikut cara mempublish website anda :

  1. Mendapatkan Domain
  2. Setting name server di Domain
  3. Mendapatkan Hosting
  4. Upload website
  5. Instal database
  6. Tes hasil
